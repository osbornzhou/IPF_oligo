# GitHub Upload Notes

This folder is the upload-ready repository payload for the BMC Genomics Research article:

**A boundary-tested transcriptomic perturbation-triage framework identifies oligonucleotide validation candidates and fibrotic disease-state markers in idiopathic pulmonary fibrosis**

## Included

- `README.md`
- `scripts/`
- `metadata/`
- `data_processed/`
- `data_external/`
- `results/`
- `manuscript/`
  - submission DOCX/PDF
  - main figure assets
  - editable SVG figure sources
  - Additional files 1-14
  - reproducibility package

## Excluded

- `data_raw/`

The raw GEO download cache is approximately 24.8 GB and is not appropriate for a normal GitHub repository. Raw source data are public and recoverable from the GEO accessions listed in the manuscript and Additional file 1.

## Suggested upload method

Use GitHub Desktop, command-line Git, or the GitHub web interface to upload the contents of this folder to:

https://github.com/osbornzhou/IPF_oligo

For long-term citation, create a GitHub release after upload and archive the release with Zenodo to obtain a DOI.
